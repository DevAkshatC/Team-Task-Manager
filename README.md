# Team Task Manager (Full Stack Backend)

## 🚀 Features
- User Authentication (JWT)
- Role-Based Access (Admin / Member)
- Project Management
- Task Assignment & Tracking
- Dashboard (task stats, overdue detection)

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 🌐 Live URL
https://your-app.up.railway.app

## 📦 API Endpoints

### Auth
POST /api/auth/signup  
POST /api/auth/login  

### Projects
POST /api/projects (Admin only)

### Tasks
POST /api/tasks (Admin only)  
GET /api/tasks  
PUT /api/tasks/:id  

### Dashboard
GET /api/dashboard
